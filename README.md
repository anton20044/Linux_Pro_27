Лабораторная работа №27 по курсу Linux Pro.

Задание:

 Реализовать nginx + php-fpm (wordpress) + python (django) + js(react) с использованием разных портов (8081, 8082, 8083).

Решение:

Подготовлен плейбук Ansible, который конфигурирует виртуальную машину.
На виртуальной машине развернут docker, в котором запущено 4 контейнера:
 - nginx
 - php-fpm
 - python
 - js

На порту 8081 запущен django (см скриншот 8081)

На порту 8082 запущен js (см скриншот 8082)

На порту 8083 запущен wordpress (см скриншот 8083)


Разворачивание ВМ и их настройка с помощью vagrant и ansible.
